API Configuration

Use your API key to validate licenses from your Roblox game.

Generate your first API key to get started
POST https://api.venonroblox.com/validate
POST https://api.venonroblox.com/create
GET https://api.venonroblox.com/licenses

Example Usage (Roblox Lua)

local HttpService = game:GetService("HttpService")
local API_URL = "https://api.venonroblox.com/validate"
local API_KEY = "your_api_key_here"

function validateLicense(licenseKey)
    local response = HttpService:PostAsync(API_URL, HttpService:JSONEncode({
        apiKey = API_KEY,
        license = licenseKey,
        hwid = game:GetService("RbxAnalyticsService"):GetClientId()
    }), Enum.HttpContentType.ApplicationJson)
    
    local data = HttpService:JSONDecode(response)
    return data.valid
end
